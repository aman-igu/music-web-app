# 🎵 Melodix

A full-stack music streaming platform built using the MERN stack that enables users to stream music, manage playlists, and interact through an AI-powered chatbot.

---

## ✨ Features

-  Secure JWT Authentication
-  Artist & Listener Roles
-  Music Upload & Streaming
-  Playlist Management
-  Song Search
-  AI Chatbot Integration
-  Responsive User Interface

---

## 🛠 Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT (JSON Web Tokens)

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/aman-igu/music-web-app
```

Install dependencies

```bash
npm install
```

Start the backend

```bash
node server.js
```

Start the frontend

```bash
cd frontend
npm install
npm run dev
```
